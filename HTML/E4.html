URL
Dirección única utilizada para acceder a recursos en internet, como páginas web, archivos, imágenes o documentos
HTTP
Código que se establece para que el computador solicitante y el que contiene la información solicitada puedan “hablar” un mismo idioma a la hora de transmitir información por la red
TCP/IP
Conjunto de reglas estandarizadas que permiten a los equipos comunicarse en una red como Internet
Internet
Sistema de redes interconectadas mediante distintos protocolos que ofrece una gran diversidad de servicios y recursos
W3C
Organización internacional que tiene como objetivo crear estándares para la World Wide Web
HTML
Lenguaje de marcado utilizado en la creación de páginas web
